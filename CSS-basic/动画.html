CSS Animation

linear 线性（匀速）
ease 逐渐加速（慢快慢）
ease-in 开始慢，逐渐加快
ease-out 开始快，逐渐减慢
ease-in-out 慢→快→慢（更柔和）

Note: animation-fill-mode: forwards 让动画结束后保持终点样式（当前代码未启用）

animation-direction: alternate-reverse 让动画在每次循环时反向播放（先反向，再正向，循环）

animation-duration: 4s; 动画单次持续时间（移动变色动画用4秒，缓动对比用7秒）
animation-delay: 0.5s; 动画延迟运行时间（移动变色动画延迟0.5秒启动）
